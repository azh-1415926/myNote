# **`Markdown`标记语言**
## `Markdown`简介
* ### `Markdown` 概要
    * *`Markdown` 语言在 2004 由约翰·格鲁伯(John Gruber)创建*
    * *`Markdown` 是一种轻量级标记语言*
    * *`Markdown` 支持图片、图表、数学式，兼容 `HTML` 标签语言*
    * *`Markdown` 编写的文档可以导出 `HTML` 、Word、图像、PDF、Epub 等多种格式的文档*
    * *`Markdown` 编写的文档后缀为 `.md`, `.markdown`*

* ### `Markdow` 应用
    * *许多网站广泛使用 `Markdown` 来撰写帮助文档或是用于论坛上发表消息。 如 GitHub、Reddit、Diaspora、Stack Exchange、OpenStreetMap 、SourceForge、简书等*  
    * *可用于撰写电子书*

## **一、`Markdown`标题**
> ### **`Markdown` 一、二级标题**
> * *可用 = 和 - 标记*
> * **= 和 - 的注意事项**
>   * *=或-应在后面另起一行使用*
>   * *=或-的数量没有明确的要求*
>       * *如：一级标题用=*
>       * *如：二级标题用-*
> ### **`Markdown` 1-6级标题**
> * *可用#来标记1-6级标题*
> * **#的注意事项**
>   * *#应放在标题文本的开头，需用空格隔开*
>   * *#的数量表示标题的级别*
>       * *如一级标题用1个#*
>       * *如二级标题用2个#*
>       * *...*
>       * *如六级标题用6个#*
* ### **一级标题**
    * *在后一行使用1个以上=*
```markdown
    一级标题
    ===
```
* ### **二级标题**
    * *在后一行使用一个以上-*
```markdown
    二级标题
    ---
```
* ### **1-6级标题**
    * *在开头使用1到6个#号,并用空格隔开*<br>
```markdown
    # 1级标题
    ## 2级标题
    ### 3级标题
    #### 4级标题
    ##### 5级标题
    ###### 6级标题
```
* * ***4-6级标题展示:***
***
* * #### 4级标题
* * ##### 5级标题
* * ###### 6级标题

## **二、`Markdown`段落格式**
> ### **`Markdown` 换行**
> * *使用2个以上空格加回车*
>   * *即在行末的文本后空两个空格*<br>
> * *使用 `HTML` 的`<br>`标签*(**推荐**)
> ### **`Markdown` 字体**
> * *使用一对 * 或者 _*
>   * *嵌套在在文本两边*
>   * *使用时应紧挨着文本标记，如`*字体*`*
>   * *数量对应相应的字体*
>       * *1个 * 或 _ 标记斜体字体*<br>
>       * *2个 * 或 _ 标记粗体字体*
>       * *3个 * 或 _ 标记斜粗体字体*
> ### **`Markdown` 分割线**
> * *用在文本行后一行*
> * *使用3个以上的 * 或者 - 来标记*<br>
>   * *容易与标记标题的 - 混淆，故*
>       * *当分割线所在行的上一行为文本且定义了标题级别、为引用、为代码区块等用---，若不满足上诉情况仍想用---，请另起一空白行使用---*
>       * *当分割线所在行的上一行为为文本且未定义标题级别可用****
> ### **`Markdown` 删除线**
> * *在文本的两端加上~~*<br>
> ### **`Markdown` 下划线**
> * *用一对标签对`<u>文本</u>`来表示*<br>
> ### **`Markdown` 脚注**
> * *格式类似这样 [^脚注]:内容*
* ### **字体**<br>
    * *使用一对不同数量的 * 或者 _ 来标记*
```markdown
    *斜体*  
    **粗体**  
    ***粗斜字体***
```  
* * ***字体效果展示:***
***
* * *斜体*<br>
* * **粗体**<br>
* * ***粗斜字体***
***

*使用3个以上的 * 或者 - 来标记*
```markdown
    ***  
    * * *  
    ---
    - - -
```
***分割线展示:***
***
* ### **删除线**
*在文字的两端加上~~*
```markdown

```
***删除线展示:***

*下划线用一对标签对`<u></u>`来表示*
```html

***下划线展示:***

## **三、`Markdown`列表**
> ### **`Markdown` 无序列表**
>   * *用在文本开头*
>   * *使用 * 或 + 或 - 标记*
>   * *注意用空格隔开*
> ### **`Markdown` 有序列表**
>   * *用在文本开头*
>   * *使用数字加上 . 来标记*
>   * *也需要用空格隔开*
> ### **`Markdown` 列表嵌套**
> * *需要在所需要嵌套的子列表前面加上2个或4个空格*
* ### **无序列表**
    * *使用 * 或 + 或 - 标记,要在其后加个空格*
```markdown
    * 第一项
        * 第二项
            * 第三项
```
* * ***无序列表展示:***
***
* 第一项
    * 第二项
        * 第三项
***
* ### **有序列表**
    * *使用数字加上 . 来标记*
```markdown
    1. 第一项
    2. 第二项
    3. 第三项
```
* * ***有序列表展示:***
***
1. 第一项
2. 第二项
3. 第三项
***
* ### **列表嵌套**
    * *在子列表的前面加上2个或4个空格*
```markdown
    1. 第一项
        - 第一项嵌套
            - 第二项嵌套
    2. 第二项
        - 第一项嵌套
            - 第二层嵌套
```
* * ***列表嵌套展示:***
***
1. 第一项
    - 第一项嵌套
        - 第二项嵌套
2. 第二项
    - 第一项嵌套
        - 第二层嵌套
***
## **四、`Markdown`区块**

> * *使用 > 标记，可与列表进行嵌套*
* ### **区块**
    * *使用 > 标记*
```markdown
    > 区块引用
    > 这是区块引用
    >> 区块引用的嵌套<br>
    >> 嵌套
```
* * ***区块展示:***
***
> 这是区块引用
>> 区块引用的嵌套<br>
>> 嵌套

    * *区块中使用列表*  
```markdown
> 1.第一项<br>
> 2.第二项
>> * 嵌套的第一项
>>   * 嵌套的第二项
```  
* * ***区块嵌套列表展示:***  
***
> 1.第一项<br>
> 2.第二项
>> * 嵌套的第一项
>>   * 嵌套的第二项

* ### **列表嵌套区块**
    * *列表中使用区块*
```markdown
    * > 列表中的区块
```
* * ***下列表嵌套区块展示:***
***
* > 列表中的区块

## **五、`Markdown`代码**
> ### **`Markdown` 代码段**
> * *用一对 ` 包裹一个函数或片段的代码*
>   * *如：``` `代码` ```*
> * *若想显示 ` ` ` 也可用一对反引号包裹*
>   * *两边需要用空格隔开*<br>
>   * *对 ` 的转义*<br>
>   * *如：`` `反引号` ``*<br>
>   * *如：` `` 反引号 `` `*<br>
>   * *如：` ``` 反引号 ``` `*
> ### **`Markdown` 代码区块**
> * *在代码开头用4个空格或者1个制表符，注意代码区块的前面一行要为空*<br>
> * *可用一对 ``` 包裹一段代码,并指定语言(也可以不指定)*
* ### **代码段**
    * *段落上的一个函数或片段的代码用一对 ` 包裹起来*
```markdown
    `printf()`
```
* * ***代码段展示:***
***
`printf()`
***
* ### **代码区块**
    * *代码区块用4个空格或者1个制表符*
```
    #include<stdio.h>
    int main(){
        printf("Hello World!");
        return 0;
    }
```
* * *还可用一对```或~~~包裹一段代码,并指定语言(也可以不指定)*
```markdown
    ```c
    #include<stdio.h>
    int main(){
        printf("Hello World!");
        return 0;
    }
    ```
    ~~~c
    #include<stdio.h>
    int main(){
        printf("Hello World!");
        return 0;
    }
    ~~~
```
* * ***代码段展示:***
***
```c
#include<stdio.h>
int main(){
    printf("Hello World!");
    return 0;
}
```
~~~c
#include<stdio.h>
int main(){
   printf("Hello World!");
    return 0;
}
~~~

## **六、`Markdown`链接**
> ### **`Markdown` 链接**
> * #### **语法格式**
>   * *`[链接名称](链接地址)`*
>   * *`<链接地址>`来标记*<br>
> ### **`Markdown` 高级链接**
> * #### **语法格式**
>   * *先声明变量，如：`[链接名称][变量名称]`*
>   * *后再对变量赋值，如：`[变量名]:网址`*
* ### **链接**
    * *用[ ]括起链接名称，后面紧接着一对( )括起链接地址*
```markdown
    [百度一下](<https://www.baidu.com>)<br>
    <https://www.runoob.com>
```
* * ***链接展示:***
***
[百度一下](<https://www.baidu.com>)<br>
<https://www.runoob.com>
***
* ### **高级链接**
* *通过定义变量来设置链接，变量赋值可在文档末尾进行*<br>
    * *声明变量:*
```markdown
    [链接名称][变量名]

    [Google][1]<br>
    [Runoob][runoob]
```
* * *其后赋值:*
```markdown
    [变量名]:网址

    [1]: http://www.google.com/<br>
    [runoob]: http://www.runoob.com/
```
***高级链接展示:***
***
[Google][1]<br>
[Runoob][runoob]

[1]: http://www.google.com/
[runoob]: http://www.runoob.com/

    * *用 | 和 - 隔开*
```markdown

| 单元格  | 单元格 |

## **九、`Markdown` 高级技巧**
> ### **`Markdown` 高级技巧**
> * **支持 HTML 元素**
>   * *目前支持的 HTML 元素有：*
>       * `<kbd>`
>       * `<b`>
>       * `<i>`
>       * `<em>`
>       * `<sup>`
>       * `<sub>`
>       * `<br>`
>       * ...等
> * **转义**
>   * *`Markdown` 使用反斜杠转义特殊字符*
> * **公式**
>   * *`Markdown Preview Enhanced` 使用 `KaTeX` 或者 `MathJax` 来渲染数学表达式*
>       * *KaTeX 拥有比 MathJax 更快的性能*
>       * *KaTeX却少了很多 MathJax 拥有的特性*
>       * 可以查看 KaTeX supported functions/symbols 来了解 KaTeX 支持哪些符号和函数*
>   * **分隔符**
>       * *`$...$ 或者 \(...\)` 中的数学表达式将会在行内显示*
>       * *`$$...$$ 或者 \[...\]` 或者 ```math 中的数学表达式将会在块内显示*
* **支持 HTML 元素**
   * *一些 `HTML` 元素*
       * `<kbd>`
            * *定义键盘文本样式*
                * *将产生一个行内元素，以浏览器的默认 monospace 字体显示*
       * `<b>`
            * *定义粗体的文本*
       * `<i>`
            * *定义斜体文本*
       * `<em>`
            * *定义被强调的文本*
       * `<sup>`
            * *定义上标文本*
       * `<sub>`
            * *定义下标文本*
       * `<br>`
            * *插入换行符*
       * ...等
    * *规范用法*
        * *标题应该用 `<h1> - <h6>` 标签表示*
        * *被强调的文本应该用 `<em>` 标签表示*
        * *重要的文本应该用 `<strong>` 标签表示*
        * *被标记的或者高亮显示的文本应该用 `<mark>` 标签表示*
